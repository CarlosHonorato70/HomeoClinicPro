🛡️ Termos de Uso e Política de Privacidade

Em conformidade com a Lei Geral de Proteção de Dados (Lei nº 13.709/2018)

1. Tratamento de Dados Pessoais Sensíveis

O HomeoClinic Pro processa dados pessoais sensíveis de saúde conforme Art. 11, II, "f" da LGPD — tutela da saúde em procedimento realizado por profissional de saúde. Todos os dados são criptografados com AES-256 e armazenados localmente no dispositivo do profissional. Nenhum dado é transmitido para servidores externos sem consentimento explícito.

2. Finalidades do Tratamento

Os dados coletados são utilizados exclusivamente para: (a) prestação de assistência à saúde homeopática; (b) gestão do prontuário eletrônico conforme Resolução CFM nº 1.821/2007; (c) emissão de prescrições e documentos clínicos; (d) repertorização e análise de sintomas para tratamento homeopático. O período de retenção é de 20 anos conforme legislação vigente.

3. Direitos do Titular dos Dados

O paciente possui direito a: confirmação de tratamento, acesso aos dados, correção de dados incompletos, anonimização ou bloqueio de dados desnecessários, portabilidade dos dados (exportação), eliminação de dados (respeitando prazos legais de retenção), informação sobre compartilhamento, e revogação do consentimento.

4. Medidas de Segurança

Implementamos: criptografia AES-256 para dados em repouso; controle de acesso por autenticação; registro de auditoria de todas as operações; backup criptografado; conformidade com NGS2 (Nível de Garantia de Segurança 2) da SBIS/CFM. O encarregado de dados (DPO) pode ser contactado através das configurações do sistema.

5. Consentimentos Necessários
